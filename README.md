# Faktury

Nový slovenský fakturačný a účtovný systém pre živnostníkov, malé s.r.o. a účtovníkov.

## Cieľ

MVP 1.0:
- slovenská fakturácia
- IČO, DIČ, IČ DPH
- IBAN/BIC
- SEPA QR platby
- slovenské DPH režimy
- klienti a dodávatelia
- faktúry
- náklady
- bankové transakcie
- párovanie úhrad
- dokladový inbox
- mesačný export pre účtovníka
- audit log
